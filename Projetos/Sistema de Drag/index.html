Range syntax in CSS if() function

Requires Chrome 142+

HTML Markup

HTML 
  <div class="item"
      data-columns="1">
      ...
  </div>

CSS Source

CSS 
  .item {
    background-color: if(
    style(attr(data-columns type(<integer>)) > 2): royalblue; 
    else: dimgray
    );
  }
